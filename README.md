```mermaid
graph LR
    A[🧠 User Intent] --> B[📝 Prompt Engineering]
    B --> C[🤖 LLM / Agent]
    C --> D{Tool Calling?}
    D -->|Yes| E[🔧 External Action / API]
    D -->|No| F[📤 Structured Output]
    E --> G[🔄 Orchestration Layer]
```
